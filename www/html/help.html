Welcome to Psygraph

Psygraph keeps track of your meditation, breathing, and mindfulness through the use of various tools, such as a timer, counter, periodic gong, and note-taker. Each tool records events in response to button presses or device sensor changes, and those events are stored on your phone. You can optionally upload that information to any server with the Psygraph WordPress plugin to further analyze or visualize that event data.

Each tool has its own page. The main page is the Home page, which displays the device status. The other tools are:

  • Stopwatch: Time your meditations and track your stillness.
  • Timer: Set bells for the end of a meditation, or schedule random alarms with inspiration quotes.
  • Counter: Measure your awareness by counting your breaths.
  • Note: Track your thoughts with text and audio notes that automatically create blog posts.

To select and visualize events that you have recorded with those tools, use the Home, List, andGraph pages.

Menus

The menu bar at the top of the app has buttons that let you View a different tool, or perform an Action. Below that are three tabs for each tool: a main tab, a settings tab, and an info tab. The Action menu displays the following options:

  • About: Open the About dialog.
  • Prefs: Open the app Preferences to specify global options.
  • Login/Logout: Login or logout of the WordPress server.
  • Help: Open the Help page.


Categories

All events (such as stopwatch start/stop times) are assigned to a category. Four categories are present by default: Uncategorized, Meditate, Exercise, and Think. The current category can be changed via the tool settings tab.

New categories can be added in the application Preferences. Every category has settings that govern its behavior, such as a particular background color, sound, and set of strings that are displayed by the timer alarms. Those settings are accessed by going to the Category Prefs dialog via the application preferences. Finally, if you are uploading your data to a WordPress server, that data is tagged with its associated category.



Preferences

The Preferences page displays lets you control which tools and category are shown, wether to enable debugging, and enables connecting to Bluetooth devices (if debugging is enabled).
