Timer

The Timer tool is a countdown alarm: the top widget shows time remaining and the bottom widget shows total time. You start/stop the timer with the right button and reset the timer with the left button. When the timer goes off, an alarm (notification) displays a randomly selected message and plays a sound. The alarm sound and text are specific to each category, and are changed in the category preferences.

The settings let you:

  • Specify what happens when the timer elapses.
  • Run the timer in a loop.
  • The two widgets on the lower portion of the screen let you set the countdown duration and a random amount of time that should be added to that duration.
  • Schedule additional alarms (which may occur even when the app is closed).
