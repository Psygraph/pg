Note

The Note tool allows you create text notes and voice memos. Two text fields are displayed: one for the note title, and one for the note text. The right button records audio: the button turns red while recording, which you stop by pressing the button a second time. Notes are saved by pressing the left button. If you are online, notes and audio data are saved as a blog post.

The settings allow you to:

  • Attach a location to the note.
  • Turn on/off the submit confirmation dialog.
