Info

The Timer tool is a countdown alarm: the top widget shows time remaining and the bottom widget shows total time. You start/stop the timer with the right button and reset the timer with the left button. When the timer goes off, an alarm (notification) displays a randomly selected message and plays a sound. You then have the opportunity to indicate whether you were mindful or not.

The alarm sound and text are specific to each category, and are changed in the category preferences. The settings on this page let you:

  • Specify an alarm that happens when the timer elapses.
  • Run the timer a specified number of times.
  • Set the countdown duration and specify a random interval that is added to that duration.
